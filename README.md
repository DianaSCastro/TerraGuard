# TerraGuard
MIT HACKNATION GLOBAL HACKATON
Hi we are from México

data_layer:	Obtiene datos de GEE, Open-Meteo, Gemini y otros servicios.
processing_layer:	Calcula índices de riesgo, combina factores climáticos y geoespaciales.
ai_layer:	Genera interpretaciones automáticas con la API de Gemini (explicación de riesgos, recomendaciones).
business_layer:	Define reglas de seguros: cuándo aprobar, rechazar o sugerir acciones.
ui_layer:	Interfaz de usuario para ver mapas, riesgos y reportes.
main.py: Orquesta todo: obtiene datos → calcula riesgos → interpreta → visualiza.
